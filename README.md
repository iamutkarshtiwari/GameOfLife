## GameOfLife

To run the code -

$ javac GameOfLife.java  
$ java GameOfLife  
Welcome to Game of Life!

O X X O

Try these following test inputs -

Input A:
(Block pattern)
```
	X X
	X X
```

Output A:
```
	X X
	X X
```

Input B:
(Boat pattern)
```
	X X O
	X O X
	O X O
 ```
 
Output B:
```
	X X O
	X O X
	O X O
```

Input C:
(Blinker pattern)
 ```
 	O X O
 	O X O
 	O X O
 ```
 
Output C:
```
 	O O O
 	X X X
 	O O O
```
